XerahS Blog

Calendar-based view of the XerahS daily development blog. Click a day to open the corresponding post on GitHub.

Tip: Blog entries are stored as Markdown files in the XerahS GitHub repository. The calendar links directly to the corresponding blob/develop URLs.

To add a new entry, create a file under docs/blog/<year>/<year>-<month>/blog-<yyyymmdd>.md in the main XerahS repository and register the date in this page’s data list.
